# Anki Code Cards

Cards with beautiful code in vanilla Anki.

## Development

### Setup

```bash
# Install dependencies
bun install
```

### Dev Server

```bash
# Start server
mise start

# Stop server
mise stop

# Restart server
mise restart

# Tail logs
mise logs
```

### Testing

```bash
mise test:e2e

# Quick healthcheck (~1.3s)
mise healthcheck
```
